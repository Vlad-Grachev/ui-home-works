## Домашняя работа №1
----------------------
### Добавление строки
Чтобы добавить новую строку, нажмите кнопку «Add». Введите необходимые данные и нажмите «Ок».
- Год выхода может быть не меньше 1970, но не больше 2050
- Количество должно быть неотрицательным целым числом
- Цена должна быть положительным действительным числом

----------------------
### Удаление строки
Чтобы удалить строку, кликните один раз на любом ее поле и нажмите «Remove».

----------------------
### Редактирование строки
Чтобы отредактировать содержимое строки, дважды кликните на любом ее поле и введите нужные данные. Действуют те же ограничение, что и пункте **Добавление строки**.

----------------------
### Сохранение состояния
Чтобы сохранить текущее состояние таблицы, нажмите кнопку «Save». Если программа была закрыта без сохранения, то все изменения будут утеряны.
